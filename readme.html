  3. Magalhaes B, Hines M, Sterling T, Schurmann F (2019) Exploiting Flow Graph of System of ODEs to Accelerate the Simulation of Biologically-Detailed Neural Networks 2019 IEEE International Parallel and Distributed Processing Symposium (IPDPS) :176-187 http://dx.doi.org/10.1109/IPDPS.2019.00028

    are available at the authors github repository

    https://github.com/BlueBrain/neurox

    A command line flag activates the functionalities of those papers. To be specific the papers are activated by these flags

    • Cache-efficiency and async. execution described on "Fully-Asynchronous Cache-Efficient Simulation of Detailed Neural Networks"

      activated by "-A 0" or "--synchronizer 0"

    • "Branch-Parallel Simulation of Detailed Neuron Models"

      • activated with "-B" or "--branch-parallelism"

    • "Exploiting Flow Graph of System of ODEs to Accelerate the Simulation of Biologically-Detailed Neural Networks"

      activated by "-G" or "--graph-parallelism"

    • "Fully-Asynchronous Fully-Implicit Variable-Order Variable-Timestep Simulation of Neural Networks"

      activated by flags "-A 0 -i 0"
